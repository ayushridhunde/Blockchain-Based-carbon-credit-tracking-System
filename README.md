# 🌱 AI-Based Carbon Credit Blockchain System

A decentralized and transparent platform to track carbon offsets using **Ethereum Blockchain** and **Streamlit**. This project ensures that carbon credits are immutable and verifiable, preventing data tampering and "greenwashing."

## 🚀 Key Features
- **Immutable Ledger:** Every transaction is recorded on a private Ethereum blockchain (Ganache).
- **Secure Authentication:** Multi-user login system for Admins and Interns.
- **Data Analytics:** Real-time visualization of carbon credit distribution using Bar Charts.
- **Footprint Calculator:** Built-in tool to estimate CO2 emissions and recommended credits.
- **Exportable Reports:** Download transaction history as a CSV file for auditing.

## 🛠️ Technical Stack
- **Frontend:** Streamlit (Python)
- **Blockchain:** Solidity (Smart Contracts)
- **Local Provider:** Ganache
- **Backend Bridge:** Web3.py
- **Data Handling:** Pandas & YAML
